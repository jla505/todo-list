## About The Project

A Todo list that does the following 
* view all projects
* view all todos in each project (probably just the title and duedateâ€¦ perhaps changing color for different priorities)
* expand a single todo to see/edit its details
* delete a todo
* Web Storage API localStorage

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps

### Prerequisites

Install npm latest 
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Setup localStorage at [https://developer.mozilla.org/en-US/docs/Web/API/Web_Storage_API/Using_the_Web_Storage_API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Storage_API/Using_the_Web_Storage_API)
2. Clone the repo
   ```sh
   git clone https://git@github.com:jla505/todo-list.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
